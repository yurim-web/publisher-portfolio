## 📁 프로젝트 구조

```
src/
├── 🖼 assets/          # 이미지, 아이콘 리소스
├── 🧩 common/          # 공용 컴포넌트 (버튼, 박스, GitHubLink)
├── 🎛 contexts/        # React Context (테마, 섹션 상태)
├── 📊 data/            # 데이터 파일 (skillsData, portfolioData)
├── 🪝 hooks/           # 커스텀 훅 (useScrollAnimation, useHeaderScroll)
├── 📄 pages/           # 페이지 컴포넌트
└── 🎨 styles/          # CSS 스타일
```
